# StackCraftLabs – AWS Terraform Modules

A collection of production-ready and tested **Terraform modules and AWS infrastructure templates**, built and maintained by **StackCraftLabs**.  
This repository focuses on reusable, well-structured IaaC Components for common AWS architectures including networking, computing, serverless, IAM, databases, and CI/CD foundations.

---

## Features

- Reusable **Terraform AWS modules**
- Example architectures for real-world cloud deployments
- AWS networking (VPC, subnets, NAT, routing)
- Compute modules (ECS Fargate, Lambda)
- Serverless stacks (API Gateway + Lambda)
- Storage (S3), Databases (DynamoDB / RDS)
- IAM best-practice blueprints
- Local testing support via **LocalStack**

---

## Repository Structure
Each Template is created in separate folder you can download the repo or download the template from release.
